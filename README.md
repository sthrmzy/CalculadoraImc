# Calculadora IMC 🧑‍⚕️💪

Este é um projeto simples de uma calculadora de IMC (Índice de Massa Corporal) em uma página web. O objetivo do projeto é fornecer uma ferramenta fácil e rápida para calcular o IMC de uma pessoa a partir de seu peso e altura.

## ⚙️ Como funciona?

A calculadora permite ao usuário inserir seu peso e altura e, em seguida, calcula o IMC automaticamente. O IMC é uma medida utilizada para avaliar se uma pessoa está com peso ideal, abaixo ou acima do peso.

A fórmula usada é a seguinte:

IMC = peso (kg) / (altura (m))²

Com base no valor do IMC, o usuário recebe uma classificação do seu estado de saúde:

- **Abaixo do peso** 🏃‍♂️
- **Peso ideal** ⚖️
- **Acima do peso** 🍔
